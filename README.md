# Notes App



## Features

* Login Screen
* Recently updated notes appear first.
* Edit notes
* Delete notes
* Smooth fade-in animations
* Fully deployed database and app


### Tech Stack

* ReactJS
* JSON Server
* Vercel/Heroku
